Preliminaries

2.1 Submanifolds

Often in mathematics there is an obvious notion of a “subobject”: given a structure on a set there is a simple way to restrict it to a subset, such that the subset can be said to have the same structure. For example, the structure on a group is the identity element, inversion, and multiplication; if there is a subset containing the identity element and which is preserved under inversion and multiplication, then we have a subgroup. Or for a topological space X, any subset A has a topology given by intersection of open sets of X with A. The structures on the subset can often be characterised by the fact that they make the inclusion map A↪X a structure preserving map (group homomorphism and continuous map respectively for the two examples).

In the case of manifolds however the definition of a submanifold is not as trivial and there are several notions that have strong claims for the title. To complicate matters, the notion that is the most widely taught (embedded submanifolds) is not the one that is most appropriate for Lie group theory. Let us begin by summarising the common definitions.

Definition 2.1. [War83, Def 1.27, Rem 1.33],[Sha97, Defs 1.1.36, 1.1.40, 1.2.10, 1.2.21]
Let ϕ : N → M be a smooth map of manifolds.

  1. ϕ is called an immersion if dϕp : TpN → Tϕ(p)M is injective at every point p ∈ N. The pair (N,ϕ) is called an immersed manifold in M.
  2. If ϕ is an injective immersion then the pair is called an immersed submanifold.
  3. Two immersed submanifolds (N1,ϕ1) and (N2,ϕ2) are called equivalent if there is a diffeomorphism φ : N1 → N2 such that ϕ1 = ϕ2 ∘ φ.
  4. If an injective immersion ϕ has the property that for every smooth map f : S → M with f[S] ⊂ ϕ[N] the map ϕ−1 ∘ f : S → N is smooth, then we call ϕ a weak embedding and the pair a weakly embedded submanifold.
  5. If an immersion ϕ is a homeomorphism from N to ϕ(N), the latter with the subspace topology of M, then we call it an embedding and the pair is called an embedded submanifold.
  6. A continuous function between Hausdorff spaces is called proper if the preimage of a compact set is always a compact set. A proper immersion submanifold is called a proper submanifold.
  7. When we say submanifold, we generally mean a weakly embedded submanifold.

This is a lot of terminology, but it harmonises the definitions in Sharpe and Warner, see below table. It is in fact a strict hierarchy: each type of submanifold is a subtype of the previous.

To return to the discussion of the first paragraph, restriction vs inclusion, these definitions are given in terms of a smooth map into M, which is to say the perspective of inclusion. So the question from the perspective of restriction is whether the manifold structure of N is determined or can be recovered solely from the image ϕ[N]. There is a simple example that shows that not even the topology of N is determined for an immersed submanifold: consider the subset {x2 = y2}⊂ ℝ2. We can split this into a line and two rays in two ways. Therefore immersed submanifolds must always be given with the immersion ϕ.

On the other hand, by using the implicit function theorem the preimage of a regular value is an embedded submanifold. The idea is to take the construction of charts in the implicit function theorem and generalise it to weakly embedded submanifolds.

Definition 2.2. [Sha97, Def 1.2.1,1.2.2,Thm 1.2.7]
Any any subset A let C(A,x) be points that can be reached from x by a continuous path through A. Let N′⊂ M be a subset. Given a chart φ : U ⊂ M → ℝm we say that W = C(N′∩ U,x) is a flat plaque if

φ(W) = φ[U] ∩ (ℝn × 0) ⊂ ℝn × ℝm−n,

an open subset of an n-dimensional affine subspace of ℝm. In that case φ|W is called a plaque chart of W. The submanifold topology on N′ is the topology such that flat plaques are open. In general the submanifold topology is finer (has more open sets) than the subspace topology [Sha97, Def 1.2.4].

Some commentary before we slot this into our hierarchy. The definition of submanifold in [Sha97] requires a collection of charts (Uα,φα) of M covering N′¯ such that the connected components of Uα ∩ N′ are all flat plaques. This has the effect of excluding the following example. Consider N as a countable collection of spheres and take ϕ : N → ℝ3 to be the map that embeds the kth sphere as ∂B(k−1,0.5(k + 1)−2); one has a sequence of non-overlapping spheres decreasing in size with the origin as a limit point. In particular any neighbourhood U of the origin must contain a sphere as a connected component of U ∩ N′ = ϕ[N], but a sphere cannot be a flat plaque. Sharpe calls this an ‘odd limit point’, but it seems no more odd than a collection of lines immersed as {n−1}× ℝ ⊂ ℝ2, which is allowed by the definition. If one truly wishes to avoid odd limits, an embedded submanifold is the correct notion. Instead, it would be better to say that Sharpe is concerned with foliations, and this shrinking chain of spheres cannot be part of a regular foliation. For embedded submanifolds the subspace and submanifold topologies coincide [Sha97, Prop 1.2.9]. The example of the dense wrapping of the line around a torus is the classic example of a submanifold in Sharpe’s sense that is not embedded.

It is also important that we use paracompact manifolds,which is equivalent to each connected component being second countable. This is because a subset of M need not be second countable in the submanifold topology. Consider the example of the irrational numbers in the real line. The flat plaques are the sets {x} for any irrational x. Thus the submanifold topology is the discrete topology, which is not second-countable.

Theorem 2.3. [Sha97, Thm 1.2.7][KSM93, Lems I.2.15–17]
Let N′⊂ M be a subset. Suppose for every point x ∈ N′ there is a chart making C(N′∩ U,x) is a flat plaque of dimension n. Let N be the set N′ with the submanifold topology and a smooth atlas comprised of the plaque charts. Then N is a manifold and the inclusion map is a weak embedding. Conversely, if ϕ : N → M is a weak embedding then this reconstructs the atlas of N.

Proof. Because we have weakened the conditions, the proof in [Sha97, Thm 1.2.7(i)] no longer works. I am not completely convinced by [KSM93, Lems I.2.15–17] because it uses a Riemannian metric, but the usual proof that a Riemannian metric induces the topology of a space already assumes that you have a topological manifold. The arguments below attempt to add fill out the details on their idea.

A subspace of a Hausdorff space is Hausdorff, and the submanifold topology is finer than the subspace topology. Hence N is Hausdorff too. We can argue in the same way for the other separation axioms, and that N is locally compact.

To show that N is paracompact, we need to show that each connected component N0 ⊂ N is second countable. Necessarily N0 is contained in a connected component M0 ⊂ M, which is second countable since M is paracompact. We apply Urysohn’s metrization theorem, which says that M0 is metricizable, to get a distance function d. Now N0 is a metric space using the restriction of d, but this induces the subspace topology, not necessarily the submanifold topology. Instead we use the intrinsic distance function d′, which is the infimum of path lengths in N0 measured with d.

In general, d′ > d, and we may worry that although N0 is path connected, some of the paths might be infinitely long, and hence N0 would not be a metric space. To see that this is not the case, for any point x ∈ N0 consider first a flat plaque W ∋ x. We use φ to move everything to ℝm. Shrinking the chart if necessary, we can assume that U is convex. d induces a distance function on ℝn ⊃ W. Therefore the intrinsic distance between any points of W is finite, because the length of a chord is an upper bound. A path between two points of N0 is compact, and therefore covered by finitely many ‘convex’ flat plaques. The triangle equality now ensures that the path length is finite.

In fact for any open set V of N0, choose any point x ∈ V . We consider the convex flat plaque centered at x from the previous paragraph. The set V ∩ W is an open subset of the affine subspace, so there is an open set V ~ ⊂ U ⊂ ℝn with V ~ ∩ W = V ∩ W. Because d generates the topology of U, there is some ball Bd(x,r) ⊂V ~. By a standard argument, any path that leaves Bd(x,r) ∩ W has length at least r, so Bd′(x,r) ⊂ Bd(x,r) ∩ W. Hence we have found an open ball with respect to d′ centered at x contained in V . As we can do this for every point of V , d′ induces the submanifold topology.

Lastly,  MSE tells us that a connected locally-compact metric space is second countable. By definition, the plaque charts form a smooth atlas. Therefore we have shown that N0 is a manifold.

That this is a weak embedding is [KSM93, Thm 1.2.7(iii)].

The converse follows from [KSM93, I.2.15 Lemma] (the construction can be carried out on the image of a weak embedding) and [Sha97, Lemma 1.1.41] (a weak embedding is compatible with a unique manifold structure). □

Finally, by [Sha97, Thm 1.2.11] proper submanifolds are automatically embedded, so we have a strict hierarchy of conditions. The standard example of an embedded submanifold that is not proper is (0,1) ⊂ ℝ. We see that a sequence in N may converge to a point of M ∖ N.

These constructions answer the question of how to endow a subset with a manifold structure. There is the possibility however that there are different constructions. Warner addresses these concerns with the following theorem:

Theorem 2.4. [War83, Remark 1.33]

  1. Let M be a manifold and A a subset of M. Fix a topology on A. Then there is at most one manifold structure on A such that (A,ϕ) is an immersed submanifold of M, where ϕ is the inclusion map.
  2. Again let A be a subset of M. If A with the subspace topology has a manifold structure such that (A,ϕ) is an immersed submanifold of M, then it is the unique topology on A such that there exits a manifold structure for which (A,ϕ) is an immersed submanifold of M.

We generalise Part (b) slightly below. This generalisation is stronger than [Sha97, Lem 2.1.41], because we only require the inclusions to be injective immersions, not weak embeddings.

Theorem 2.5. Let A be a subset of M. If A with the submanifold topology has a manifold structure such that (A,ϕ) is an immersed submanifold of M, then it is the unique topology on A such that there exits a manifold structure for which (A,ϕ) is an immersed submanifold of M.

Proof. First we address the relationship between the manifold structure on A and the plaque charts. For any function A↪M that is an immersion at p the constant rank theorem gives us a plaque chart of A at p. Applying this to the inclusion ϕ and at every point we get that A is a weakly embedded submanifold. By Part (a) of the previous theorem this is the unique manifold structure on A with the submanifold topology (the manifold structure supposed in this theorem and the plaque charts belong to the same maximal atlas).

Now we move on to the statement in the theorem. For clarity, let B the manifold that has the same points as A but a different topology and manifold structure, and let its inclusion be ψ : B↪M. In terms of points, the composition ϕ−1 ∘ ψ : B → A is the identity map, so bijective. We know that (A,ϕ) is a weak embedding. Hence ϕ−1 ∘ ψ is in fact a smooth map of manifolds. It must also be an immersion: if there is a smooth path α in B such that ϕ−1 ∘ ψ ∘ α represents the zero vector then ψ ∘ α represents the zero vector in M by applying ϕ and so α represents the zero vector using that ψ is an immersion. At the final step of the proof of Part (b) of the above theorem, Warner uses his Exercise 1.6 to conclude that a smooth bijective immersion is a diffeomorphism. We do the same. □

2.2 Lie Brackets and Distributions

It is common in a course on manifolds to study vector fields and their integral curves. The key local result is

Theorem 2.6 (Picard-Lindelöff). [Sha97, Theorem 2.1.1][IL, Theorem 1.2.1]
Let F : J × U ⊂ ℝ × ℝn → ℝn be a smooth time-dependent vector field. Assume 0 ∈ J. Consider u : ℝ → U the system of ODEs u′(t) = F(t,u(t)). For any c ∈ U there exists a 𝜀 > 0 such that there is a unique solution smooth solution on (−𝜀,𝜀) with u(0) = c. Moreover, for any p ∈ U there is an open neighbourhood p ∈ V ⊂ U, 𝜀 > 0 and smooth map u : (−𝜀,𝜀) × V → U such that u(⋅,c) is the unique solution with initial condition c.

If one has a smooth vector field on a manifold, then we can ask for a curve γ through p = γ(0), called an integral curve, such that the tangent vectors of γ are equal to the vector field. In other words, the integral curve is a solution to the time-independent ODE γ′(t) = X(γ(t)). this theorem provides for the existence of integral curves of the vector field in every coordinate chart, and uniqueness means that they can be patched together to give unique maximal integral curves through every point.

The set of integral curves of a vector field X define a flow of the manifold: X~t(p) is defined to be equal to the integral curve of X through p at time t (ie t↦X~t(p) is the integral curve). This is a smooth function defined on a subset of ℝ × M (treating the subscript as the first variable here), in particular containing some neighbourhood (−𝜀,𝜀) × U of any point (0,p). Uniqueness of the integral curves means that this flow is reversible. Moreover, if the vector field is nonzero at a point, there is a chart containing that point where the vector field is the first coordinate vector ∂1 and the flow is translation. In the other direction, the derivative of the flow with respect to t at t = 0 gives back the vector field X.

The flow provides a way to connect points of a manifold with neighbouring points, and so can be used to define a derivative, called the Lie derivative. For example, for a function f

LXf = lim ⁡ h→0h−1 (f ∘X~ h − f ) = d dt |t=0f ∘X~t

is, by the chain rule, equal to ⟨X,f⟩. The Lie derivative of a vector field is likewise defined by

LXY = lim ⁡ h→0h−1 (dX~ −h(Y ∘X~h) − Y ) = d dt |t=0dX~−t ∘ Y ∘X~t

which is well-defined since dX~−h takes TX~h(p)M to TpM. Most importantly, the Lie derivative of a vector field is a type of commutator. This is easiest to calculate in local coordinates, specifically the ones adapted to the flow of X [Lee12, Thm 9.38]. If Y (p) = Y i(p)∂i|p then

dX~−h(Y ∘X~h)(p) = dX~−h (Y i(p + (h,0,…))∂ i|X~h(p)) = Y i(p + (h,0,…))∂ i|p LXY (p) = lim ⁡ h→0h−1 (Y i(p + (h,0,… ⁡ )) − Y i(p))∂ i|p = ∂Y i ∂x1 ∂i|p.

On the other hand, this is the commutator of X and Y

[X,Y ] = ∂Y i ∂x1 ∂i − Y i ∂1 ∂xi∂1 = ∂Y i ∂x1 ∂i.

One can check that the commutator of two vector fields is a well-defined vector field, independent of the coordinates. Therefore LXY = [X,Y ].

From this commutator formula, most of the properties of the Lie derivative of vector fields easily follow. For us, the most important are anti-symmetry [X,Y ] = −[Y,X] and the Jacobi identity [X,[Y,Z]] = [[X,Y ],Z] + [Y,[X,Z]]. There is also a generalisation of adapted coordinates to sets of linearly-independent commuting vector fields: around each point there is a chart whose coordinates vector fields are the given vector fields.

We will need a generalisation that deals with vector fields that are not commuting. To motivate why this is geometrically interesting and not just generalisation for its own sake, consider a submanifold N inside M. At each point p ∈ N we can consider the vector subspace TpN ⊂ TpM. At least locally, we can describe these subspaces as the span of independent vector fields. The natural question is the converse: given a set of independent vector fields on M, does there exist a submanifold N whose tangent space is their span? For a single vector field, the answer is affirmative, namely the integral curve.

Definition 2.7. [Sha97, 2.2.1,.2.2.2,2.3.2]
An r-dimensional distribution D on M is a choice of an r-dimensional subspace Dp of TpM at every point p ∈ M. It is called smooth if every point has a neighbourhood and smooth vector fields {X1,…,Xr} that span the subspaces. This is called a local basis for D. Equivalently, an r-dimensional distribution is an rank r subbundle of TM.

A set of vector fields is called algebraically involutive if their Lie brackets are contained in their span. Given two sets of vector fields with the same span, one is algebraically involutive if and only if the other is. Therefore algebraically involutive is a property that is defined for distributions.

A distribution is called integrable if every point has a coordinate neighbourhood in which the distribution is spanned by coordinate vector fields.
A connected r-dimensional immersed submanifold N is called an integral submanifold of D if at every point TpN = Dp.

Consider the example of nested spheres centered at the origin in ℝ3 ∖{0}. Their tangent planes define a 2-distribution. In a neighbourhood of any point, suitable polar coordinates show that it is smooth and integrable. However there are not global vector fields spanning this distribution, due to the hairy ball theorem (every vector field on a sphere vanishes at least once). By definition, any of the spheres is an integral manifold of this distribution.

The important example of a non-integrable distribution is given in ℝ3 by the planes normal to the vector field n(x,y,z) = (y,−x,1) [Sha97, Ex 2.3.1]. There can be no integral manifold containing the origin. Suppose there were. Because the tangent plane at the origin is the xy-plane, we know that there is a neighbourhood where is a graph over x and y. Now take a small loop in the integral manifold around the origin lying in this neighbourhood. It’s height is monotonic, it doesn’t close up. This is a contradiction. A local basis at the origin is given by X1 = ∂x − y∂z, X2 = ∂y + x∂z. Observe that [X1,X2] = 2∂z, so that it is not algebraically involutive.

If a distribution is integrable, then every point has an integral submanifold through it, just by taking a coordinate plane in an appropriate chart. And if there is an integral submanifold of the distribution at a point, the local basis of the distribution gives a vector field on it. So then the Lie bracket is again tangent to the integral submanifold, showing that the vector fields are algebraically involutive at that point. Hence algebraic involutive is a necessary condition to the existence of an integral submanifold.

Theorem 2.8 (Frobenius’ theorem). [Sha97, Thm 2.4.1, Cor 2.4.2][War83, Thm 1.60, 1.62][Lee12, Thm 19.2]
A distribution is integrable if and only if it is algebraically involutive. An integral submanifold is weakly embedded.

Proof. It suffices to show that locally every distribution can be spanned by linearly-independent commuting vector fields. If it can be, then by the discussion above there is an adapted coordinate system, and the connected component of the integral submanifold in this chart through a given point is a coordinate plane, thus a flat plaque.

Since this is a local question, we will work in a local coordinate. Let D be rank r and take any local basis X1,…,Xr. By rotating coordinates, assume that at p the basis is complementary to ∂r+1,…,∂m. Let π be the projection onto the first r coordinates. dπ|D : D → TU is a smooth bundle homomorphism, and it is an isomorphism at p by our set-up. Hence there is a neighborhood of p where it is invertible. Hence Y i = (dπ|D)−1∂i is also a local basis of D at p. But by the naturality of the Lie bracket

[Y i,Y j] = [(dπ|D)−1∂ i,(dπ|D)−1∂ j] = (dπ|D)−1[∂ i,∂j] = 0. □

We can use uniqueness to unite integral submanifolds of a distribution into their maximal connected components. Each of these is called a leaf and the collection of leaves is a foliation.

Perhaps let us revisit the example from the previous section of a weakly embedded submanifold that is not a submanifold in the sense of Sharpe, the shrinking spheres. There is no distribution on ℝ3 with this as an integral submanifold. If we take a sequence of corresponding points on the spheres, pn = cn + rnω for ω ∈ 𝕊2 for cn,rn the centers and radii of the spheres, the limit is the origin. But all the tangent planes Tpn(∂B(cn,rn)) are parallel, so their limit is just any of them translated to the origin. Taking several such sequences, we see that the distribution cannot be continuous at the origin.

2.3 Covers

In this section we summarise covering maps. To avoid some technical topological issues, we only give the definition in the smooth case.

Definition 2.9. [?, p. 56] Let X and X~ be smooth manifolds and π : X~ → X a smooth surjection. An open set V ⊂ X is called evenly covered if π−1[V ] is a disjoint union of open sets, each diffeomorphic to V under π. The map π is called a covering if every point has an evenly covered neighbourhood. The connected components of an evenly covered set are called sheets, and the preimage of a point of Y is called its fibre. X is called the covering space and Y the base space.

It is easy to see that every covering is a surjective local diffeomorphism, but the converse is not true.

Example 2.10. Consider γ : (−2π,2π) → 𝕊1 given by t↦eit. This is a surjective local diffeomorphism. But the point 1 ∈ 𝕊1 does not have an evenly covered neighbourhood. To prove this, notice that open subsets of evenly covered sets are themselves evenly covered. So if there exists an evenly covered neighbourhood, there exists one of the form V = {eit∣t ∈ (−𝜀,𝜀)}. But

γ−1[V ] = (−𝜀,𝜀) ∪ (2π − 𝜀,2π) ∪ (−2π,−2π + 𝜀).

Clearly (2π − 𝜀,2π) is not diffeomorphic to V under π, it is not even surjective. Therefore γ is not a covering map.

However the map γn : 𝕊1 → 𝕊1 given by z↦zn is a covering map with n sheets. And γ∞ : ℝ → 𝕊1 given by t↦eit is a covering map with infinitely many sheets.

The most important property of covering maps is that maps into the base space can often be ‘lifted’ to the covering space. A function f~ : Y →X~ is called the lift of f : Y → X if f = π ∘f~. To state the following theorem we also need the concept of homotopy. Given two functions f,g : Y → X a homotopy relative K ⊂ Y is a function H : Y × [0,1] → X such that H(y,0) = f(y) and H(y,1) = g(y) for all y ∈ Y , and H(k,t) = f(k) = g(k) for all k ∈ K,t ∈ [0,1]. We say that the paths are homotopic relative K. A typical example is if Y = [a,b] and K = {a,b}. A map f : [a,b] → X is a path into X. If it is homotopic to g relative K then there is deformation of f into g such that the endpoints of the curves are not moved.

Theorem 2.11 (Homotopy Lifting Property). [War83, Thm 3.23][?, Thm 1.7(c)]
Let π : X~ → X be a covering map. Given a map F : Y × [0,1] → X and a map F~0 : Y ×{0}→X~ lifting F|Y ×{0} there is a unique map F~ : Y × [0,1] →X~ lifting F such that F~|Y ×{0} = F~0.

Proof. As a first step, we will construct local lifts. Every point (y,s) ∈ Y × [0,1] has a connected open neighborhood Ns × (as,bs) such that the image under F is contained in an evenly covered neighborhood of F(y,s). These neighborhoods cover {y}× [0,1], a compact set, so there exists a finite subcover index by si. Set N = ∩Nsi, also open and connected, and choose a partition 0 = t0 < t1 < ⋯ < tn = 1 of [0,1] so that each [tj,tj+1] belongs to an interval (asi,bsi). Thus we have a finite cover N × [tj,tj+1] of {y0}× [0,1] such that the image of each is contained in an evenly covered subset Uj (we cannot say that the image itself is evenly covered, because it might not be open in X).

For clarity, set F~j(y) = F~(y,tj) and Fj(y) = F(y,tj). Assume inductively that we have constructed the lift F~ on Mj × [0,tj], for some neighborhood Mj ⊂ Y of y. This means π ∘F~j(y) = Fj(y), or in other words F~j(y) ∈ π−1[Fj(y)]. Let U~j ⊂X~ be the sheet of Uj that contains F~(y,tj). It might be necessary to shrink Mj, by setting Mj+1 to be the connected component of Mj ∩ (F~j|Mj)−1[U~j] that contains F~(y,tj). This guarantees that F~j[Mj+1] ⊂U~j. Now we extend the definition of F~ by using the local inverse π−1 : Uj →U~j:

F~(y′,t′) = π−1 ∘ F(y′,t′) for y′∈ M j+1,t′∈ [t j,tj+1].

The base case of the induction j = 0 is trivial: we set M0 = N and already have F~0 given to us in the theorem.

Next we prove uniqueness by adding to the induction proof above, in particular we will use the thus-far unused property of connectedness of Mj. Suppose that F~′ is another lift of F fitting the conditions of the theorem. Suppose that we have already established that F~ and F~′ are equal on Mj × [0,tj]. Because Mj+1 × [tj,tj+1] is connected, both F~ and F~′ must map it to the same component of U~j. But on U~j, π is injective. Hence the lifting property π ∘F~ = F = π ∘F′~ implies the equality F~ = F~′ on Mj+1 × [0,tj+1] ⊂ (Mj × [0,tj]) ∪ (Mj+1 × [tj,tj+1]). Since they are equal at j = 0, the induction is complete.

Finally, we use local existence and uniqueness to patch these together into a global function. □

Note that F~ is define essentially as the composite of π−1 and F on suitable patches. In particular, this shows us its regularity. For instance, if π and F are smooth then so is F~.

There are two special cases of this theorem that are used more than any others. The first is the case that Y is a one point space. Then we can interpret F as a path [0,1] → X. The initial lift F~0 is nothing other than a choice of point in π−1[F(0)]. Thus every curve has a lift, unique up to choice of starting point in X~ lying over the starting point in X.

The second use is when F is the homotopy of two curves relative their endpoints. Formally Y = [0,1]. Each of the curves can be lifted with the same starting points. The fact that the homotopy can be lifted tells us that the two curves are homotopic in the covering space. In particular, they have the same finishing points.

An application is the construction of lifts of other maps.

Corollary 2.12 (Lifting Property). [?, Prop 1.33] Let π : X~ → X be a covering. Suppose that Y is simply connected. This means that any two paths in Y with the same endpoints are homotopic. Then for any map f : Y → X and any points y0 ∈ Y and x~0 ∈ π−1[f(y0)], there exists a unique lift f~ of f with f~(y0) = x~0.

Proof. For any point y ∈ Y , let γ : [0,1] → Y be a path between y0 and y. Then by the discussion above, the path f ∘ γ has a lift γ~ : [0,1] →X~ with γ~(0) = x~0. Define f~(y) = γ~(1). This is well-defined, because if there is any other path γ to y, then by assumption there is a homotopy H between the two paths. Then f ∘ H is a homotopy of f ∘ γ and f ∘ γ′. But we know, again from the discussion above, that the lifts of homotopic paths have the same finishing point. □

For any space, there is a simply connected covering space. The idea is to expand on the idea of the above corollary, that the points of the covering space can be described as the endpoints of paths.

Theorem 2.13. [?, p.63] Let X be a connected smooth manifold. Then there exists a simply connected smooth manifold X~ and smooth covering map π : X~ → X. X~ is called the universal cover of X.

Sketch of proof. Remarkably the proof is constructive. Fix a base point of X. Define the universal cover X~ to be the set of paths originating at the base point, up to homotopy equivalence. The covering map π sends a path to its endpoint. For any path γ take a simply connected open neighbourhood V of its endpoint x = π(γ) = γ(1). Then concatenation of γ with a path in U from x to any point of U is a bijection from U to a subset of X~. Declaring these bijections to be homeomorphisms makes X~ into a topological space, and declaring further that they are diffeomorphisms makes X~ into a manifold. □

The reason for the epitaph ‘universal’ is that if πX : Y → X is any covering map, and π : X~ → X is the universal cover, then the lifting property tells us that π~ lifts to a covering map πY of Y , with πX = πY ∘ π.

Corollary 2.14. The universal cover is unique in the following sense: If X~,X~′ are simply connected and π : X~ → X,π′ : X~′→ X are covering maps, then for any points x ∈ X, x~ ∈ π−1[x], and x~′∈ π−1[x] there is a diffeomorphism from X~ to X~′ that takes x~ to x~′. □

We can apply this corollary to the universal cover itself. Given two points x~,x~′∈ π−1[x] there is an automorphism of X~ taking x~ to x~′. These automorphisms are called deck transformations of π. The set of deck transformations of the universal cover acts transitively on the fibre of any point.

2.4 Eigenvalues and Weights

Eigenvalues and eigenvectors are ubiquitous in linear algebra. If we are working over ℂ, then every linear endomorphism (linear map from a vector space V to itself) has an eigenvalue (root of the characteristic polynomial) and every eigenvalue has an eigenvector Avλ = λvλ. If we can find a basis of eigenvectors, then with respect to this basis the linear operator is a diagonal matrix. In general however, there may be fewer eigenvectors than the dimension of the vector space. A standard result in linear algebra says that every matrix is conjugate to a matrix in Jordan normal form, unique up to reordering of the blocks.

Going further, we may ask what can be said of two linear endomorphisms A,B. The key observation is to consider commuting operators. If A and B commute then B preserves the eigenspaces eigenspaces of A:

(A − λI)(Bv) = B(A − λI)v = 0.

Therefore B restricts to an endomorphism on each of the eigenspaces of A. Imposing different conditions on A restricts the possible decompositions of B. For example, if A is diagonalizable (so V decomposes as the direct sum of the eigenspaces of A) then on each eigenspace of A we can choose a basis that puts B into Jordan normal form. Hence A and B can simultaneously conjugated to normal form. Or if A is diagonalizable and has no repeated eigenvalues, ie its eigenspaces are one dimensional, then these must also be eigenspaces of B. Hence A and B are simultaneously diagonalizable.

This argument can be applied inductively to a finite set {A1,…,Ak} of pairwise commuting endomorphisms. It also extends to a commuting family of operators A = span ⁡ {A1,… ⁡ ,Ak}, a linear subspace of End ⁡ (V ) such that all operators are pairwise commuting. These are effectively equivalent, since a family is pairwise commuting if and only if a basis is pairwise commuting. Likewise, v is a simultaneous eigenvector for {A1,…,Ak} if and only if it is a simultaneous eigenvector for every operator of A. The eigenvalues are not completely independent:

λv = Av = (a1A1 + ⋯ + akAk)v = a1λ1v + ⋯ + akλkv = (a1λ1 + ⋯ + akλk)v.

We understand the eigenvalue of v to be a linear function

λ : A → 𝕂,λ(A) = a1λ1 + ⋯ + akλk.

Understood in this way, it is more common to call λ a weight of the commuting family A, v a weight vector, and the set of vectors v with Av = λ(A)v the weight space [Hal15, Definition A.14].

As an aside, the descriptor “weight” should probably replace “eigen-” even in the single operator case. Consider an operator A with a 2-weight vector v and a 3-weight vector w. Then of course A(v + w) = 2v + 3w, which is a weighted sum.

We finish with an example. Let A be the set of diagonal n × n matrices. Then a basis for this family is Ai = eii, with 1 at the ith position on the diagonal. ei is a 1-weight vector of Ai while all vectors of span ⁡ {e1,… ⁡ ,êi,… ⁡ ,en} are 0-weight (in the kernel). These are all diagonal (in particular simultaneously diagonal) so there should be a basis of weight vectors. Indeed, this is just the standard basis {e1,…,en}. The weight of ei is the linear form

A = diag ⁡ (a1,… ⁡ ,an)↦ai

because Aei = ai.
